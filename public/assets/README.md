# Icon Files

For the desktop application, you'll need to add icon files:

- `icon.ico` (Windows) - 256x256 pixels
- `icon.icns` (macOS) - Multiple sizes
- `icon.png` (Linux) - 512x512 pixels

You can create these from your logo or use a service like:
- https://www.icoconvert.com/ (for .ico)
- https://iconverticons.com/ (for .icns)

For now, the app will use the default Electron icon.
