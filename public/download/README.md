# Download Directory

This folder contains downloadable files for users.

## Files

- `InvoifyBridge.apk` - Android Bluetooth Bridge app (will be placed here after building)

## Usage

Files in this directory are publicly accessible at:
```
https://yourwebsite.com/download/filename
```

For example:
```
https://yourwebsite.com/download/InvoifyBridge.apk
```

## Note

The APK file will be generated after you build the Android app project using Android Studio.
